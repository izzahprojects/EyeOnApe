# EyeOnApe

EyeOnApe is an IoT-based monkey detection system designed to help monitor monkey activity in outdoor areas.

## Features

- Monkey detection using camera technology
- Telegram notifications
- Alarm sound alerts
- Solar-powered operation
- Dashboard monitoring

## Technologies Used

- IoT
- Camera Module
- Solar Energy System
- Telegram API

## Project Type

Final Year Project (Diploma in Computer Technology - Big Data)
